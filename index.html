사용 방법

  1. 다운로드한 ZIP 파일의 압축을 풀어주세요
  2. Android Studio에서 File → Open으로 폴더를 열어주세요
  3. Gradle Sync가 완료될 때까지 기다려주세요
  4. Build → Build Bundle(s) / APK(s) → Build APK(s)를 클릭하세요
  5. 생성된 APK를 안드로이드 폰에 설치하세요
  6. 카카오톡 → 더보기 → 설정 → 테마에서 적용하세요

이 도구는 브라우저에서 모든 처리가 이루어지며, 파일이 서버로 전송되지 않습니다.
